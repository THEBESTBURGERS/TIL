## 진행상황

- GPSD 라이브러리 설치한 상황이고 두가지 방법으로 데이터를 읽을수 있다.

    ```bash
    # 1번 gpsmon 사용
    gpsmon /dev/ttyUSB0 /var/run/gpsd.sock

    # 2번 
    cgps
    ```
