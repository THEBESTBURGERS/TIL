### sol
```
print("\\    /\\")
print(" )  ( ')")
print("(  /  )")
print(" \\(__)|")
```


---

### 풀이
python에서 \는 이스케이프 문자로 역슬래시를 입력하려면 \\로 입력해줘야한다.
